# Client Manager

## Tech stack

### Frontend
- Vue.js 3
- Pinia
- TypeScript
- Axios

### Backend
- NestJS
- Axios

## Install and run the frontend:

### Install the dependencies
```bash
cd client
npm install
```

### Start the app in development mode
```bash
npm run dev
```

### Build the app for production
```bash
npm run build
```

## Install and run the backend:

### Install the dependencies
```bash
cd server
npm install
```

### Create .env file:
```
CLIENT_ID=
SOURCE_URL=
```

### Start the app in development mode
```bash
npm run start:dev
```

### Build the app for production
```bash
npm run build
```
